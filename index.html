Email Development

Along with Web development I have also developed some mass email campaigns. Starting with a good design I have been able to create HTML emails that are ready to be seen on a variety of devices. When working with HTML emails one of my largest focuses is testing, the variety of email clients and rendering engines causes a lot of issues for HTML emails. Testing with tools like Litmus.com allows you to test to see how emails will display in a many browsers to provide users an uniform email. Using those tools I have been able to learn the tricks necessary to set an email to look uniform across all platforms

Another thing that I keep in mind is many of the rules and regulations of email. Laws like the CAN-SPAM act have rules that require information to be present in messages and campaigns and those need to be taken into account when setting up email messages. Along with those rules each email provider has rules that may cause email messages to be lost in their spam filters, knowing these rules will also help get your messages to your intended recipients.

Skills and Tools: Testing Tools like Litmus, MailChimp, Constant Contact, HTML and CSS
